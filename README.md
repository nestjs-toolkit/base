## Descrição

Pacote com várias funções helpers para agilizar o desenvolvimento com [Nest](https://github.com/nestjs/nest) framework.

## Instalação

```bash
$ yarn add @nestjs-toolkit/base
```

## Teste

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```
